# Weekly Review Prompt

Run this yourself (in Perplexity or Copilot chat) at the end of each week.
Provide current DECISIONS.md, V1_PLAN.md, and the list of merged PRs as context.

---

I am doing a weekly review of the nowu framework development.

**Context I am providing:**
- DECISIONS.md: [paste content]
- V1_PLAN.md: [paste or reference]
- PRs merged this week: [list PR titles and numbers]
- Any blockers or issues: [describe]

**Please help me:**
1. Check: are we on track with V1_PLAN.md? Which steps are done, in-progress, or not started?
2. Review: are there any decisions that should be added to DECISIONS.md based on what was built?
3. Identify: any inconsistencies between the code (based on PR descriptions) and ARCHITECTURE.md?
4. Suggest: what is the most important thing to work on next week, and why?
5. Update: give me an updated SESSION-STATE.md I can commit that reflects the current state.

Be concise. Use bullet points. Flag any risks.
