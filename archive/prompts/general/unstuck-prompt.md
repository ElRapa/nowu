# "I Am Stuck" Prompt

Use when development stalls, a decision is unclear, or an agent is producing bad output.

---

I am stuck on the following in the nowu framework:

**What I was trying to do:**
[describe the goal]

**What happened:**
[describe the problem — error, wrong output, unclear decision, disagreement with architecture]

**What I already tried:**
[list what you checked: ARCHITECTURE.md, DECISIONS.md, test output, etc.]

**Context:**
- Module: [which module]
- Related files: [list files]
- Current SESSION-STATE.md: [paste or summarize]

**Please:**
1. Diagnose what the most likely root cause is
2. Suggest 2–3 concrete next actions in priority order
3. If this requires a new decision, draft the decision entry for DECISIONS.md
4. If this is a v2 concern being pulled into v1, flag it and suggest deferring
