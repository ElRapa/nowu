# Idea Note: idea-008


## Raw Signal

When multiple Claude Code sessions are open at different altitudes simultaneously, the system has no mechanism to know which altitude a session is operating at, enforce context scoping rules, prevent cross-session bleed, or help a new session orient quickly. Experienced concretely when editing vision.md caused a need to manually re-run S1 intake validation.
